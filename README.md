# Terminal Dashboard

Интерфейс панели управления терминалами и задачами, собранный на ванильном HTML + CSS + JavaScript.

## Структура проекта

```
/
├── index.html      # Основная HTML-структура
├── styles.css      # Все стили
├── script.js       # Вся логика и интерактивность
└── assets/         # SVG-иконки и изображения
```

## Запуск

1. Запустите локальный HTTP-сервер:
```bash
python3 -m http.server 8000
```

2. Откройте в браузере:
```
http://localhost:8000
```

## Функциональность

### Боковая панель (Sidebar)
- Список задач (Task Cards)
- Клик по карточке разворачивает/сворачивает её
- **Активная карточка**: показывает список агентов с их именами и статусами (Running/Idle)
- **Свернутая карточка**: показывает только цветные индикаторы (лампочки) статусов агентов
  - Зеленая лампочка = Running
  - Серая лампочка = Idle

### Терминалы
- Два терминала расположены рядом
- Вкладки для переключения между разными сеансами
- Кнопка "+" для добавления новых вкладок
- Кнопка "×" для закрытия вкладок
- **Кнопка Split**: разделяет терминал на два окна по вертикали
  - Повторный клик объединяет обратно

### Интерактивность
- Переключение между вкладками терминала
- Добавление новых вкладок
- Закрытие вкладок (с автоматическим переключением на оставшуюся)
- Разворачивание/сворачивание карточек задач
- Split view для терминалов (вертикальное разделение)

## Технологии

- Чистый HTML5
- Чистый CSS3 (без фреймворков)
- Ванильный JavaScript (без библиотек)
- Шрифт: Menlo (моноширинный)
